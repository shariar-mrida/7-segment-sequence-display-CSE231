# Seven Segment Sequence Display

A Digital Logic Design project for displaying the sequence **FE 85 CA JH 32 dU 74** on two 7-segment displays using sequential and combinational logic.

## Project Overview

This project was developed for **CSE231: Digital Logic Design** at North South University. The goal is to design and implement a circuit that converts binary state inputs into readable characters on 7-segment displays.

The project is divided into two phases:

- **Phase 1:** Design of the combinational logic circuit using truth tables, Karnaugh maps, SOP/POS expressions, and NAND implementation.
- **Phase 2:** Automation of the display sequence using flip-flops and a 555 timer clock generator.

## Display Sequence

The circuit displays the following sequence:

```text
FE → 85 → CA → JH → 32 → dU → 74
